CSCE 242: Website Applications

Assignments

Assignment 1: Basic HTML

Basic introduction to HTML including implementation of tables, images and hyperlinks. Very basic levels of formatting and stylesheets.

Assignment 2: Basic CSS

Slightly more advanced than Assignment 1 and introduces topics such as padding, navigation, background images, and more.

Assignment 3: Page Layout

Utilized flex and columns to help the website maintain a good look even while windowed. Also implemented footers and an overall more clean look.

Assignment 5: Recreate CSS Page

Created a template for the UofSC website. Utilized a lot of the skills and features I learned up until this point.

Assignment 6: JavaScript, Buttons, Functions, and more

Implented JavaScript into the program to add features such as show and hide buttons, animations and input boxes.

Assignment 7: Conditionals

Utilized JavaScript to make an age sorting tool, as well as a money meter that fills up with the amount specified.

Assignment 8: Loops

Made a running man animation using JavaScript and grasped a firmer understanding of the methods used to create the money meter.

Assignment 9: Arrays

In order to create a list of quotes that would appear one by one, and a rainbow that would appear from top to bottom with a pot of gold at the end, I made use of arrays in JavaScript.

Assignment 10: Classes

Used classes in JavaScript to make a toy store website that displays a variety of information on the toy that is being hovered over.
